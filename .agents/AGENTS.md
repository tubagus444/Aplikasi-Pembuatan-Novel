# Aturan Proyek AetherScribe

Proyek ini mengelola aturannya secara terpusat pada file `claude.md` dan direktori `.claude/rules/`. 
Sebagai agen AI (Gemini/Antigravity), kamu **WAJIB** mematuhi instruksi berikut saat beroperasi di repositori ini:

1. **Baca Aturan Inti:** Patuhi semua aturan arsitektur, database (Dexie), dan konvensi lintas-fitur yang tertulis lengkap di `claude.md` di root proyek.
2. **Baca Aturan Per-Fitur:** Jika kamu mengerjakan atau menyentuh area fitur spesifik (seperti alur AI, Lore, Codex, Timeline, Atlas, dll), kamu **WAJIB** membaca file rule spesifiknya yang ada di dalam direktori `.claude/rules/*.md` sebelum mulai melakukan perubahan kode. (Lihat tabel indeks di `claude.md` untuk mengetahui pemetaan fiturnya).
3. **Cek Tracker:** Selalu perhatikan dan patuhi tracker aktif seperti `RENCANA-AUDIT-KODE.md`, `RENCANA-OPTIMASI-AI.md`, `RENCANA-FITUR-WORLDBUILDING.md`, dan terutama `RENCANA-AUDIT-FABLE.md`. Perbarui status di tracker tersebut jika ada perbaikan atau temuan baru yang relevan dengan instruksi pengerjaan.
